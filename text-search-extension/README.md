# 文本搜索助手浏览器插件

一个智能的浏览器插件，让您在阅读网页时可以轻松搜索选中的文本，而不需要离开当前页面。

## 功能特点

- **自动搜索**: 在任意网页上用鼠标选中文本后，会自动弹出搜索窗口显示搜索结果
- **独立窗口**: 搜索结果在独立的小窗口中显示，不影响当前阅读内容
- **多搜索引擎**: 支持Google、百度、必应等搜索引擎切换
- **智能定位**: 搜索窗口会根据鼠标位置智能定位，避免遮挡重要内容
- **易于关闭**: 点击关闭按钮或页面其他区域即可关闭搜索窗口

## 安装方法

### Chrome/Edge浏览器

1. 下载或克隆此项目到本地
2. 打开浏览器，访问 `chrome://extensions/` (Chrome) 或 `edge://extensions/` (Edge)
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹 `text-search-extension`
6. 插件安装完成！

## 使用说明

1. 在任意网页上浏览内容
2. 用鼠标选中您想要搜索的文本
3. 搜索窗口会自动弹出，显示搜索结果
4. 点击窗口右上角的 × 按钮或页面其他区域关闭窗口

## 配置选项

点击浏览器工具栏中的插件图标，可以打开设置页面：

- **搜索引擎**: 选择您偏好的搜索引擎（Google、百度、必应）
- **自动搜索**: 启用或禁用自动搜索功能

## 项目结构

```
text-search-extension/
├── manifest.json          # 插件配置文件
├── content.js            # 内容脚本，监听选中文本

├── search.js             # 搜索脚本
├── search.css            # 搜索页面样式
```

## 技术实现

- **Manifest V3**: 使用最新的Chrome扩展API
- **Content Scripts**: 在页面中注入脚本监听选中文本事件
- **IFrame**: 使用iframe显示搜索结果，确保隔离性
- **Storage API**: 使用本地存储保存用户设置
- **响应式设计**: 搜索窗口会根据屏幕大小智能定位

## 注意事项

1. 由于浏览器的CORS策略，搜索结果通过iframe显示
2. 某些网站可能会阻止iframe嵌入，此时可以点击"在新标签页中打开"
3. 插件需要访问所有网站的权限才能正常工作
